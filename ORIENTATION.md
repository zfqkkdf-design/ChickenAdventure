# Поддержка ориентаций экрана

Проект поддерживает как вертикальную (портретную), так и горизонтальную (альбомную) ориентацию экрана.

## Настройки проекта

В `project.godot` настроено:
- `window/handheld/orientation=10` - поддержка обеих ориентаций
- `window/stretch/mode="viewport"` - адаптивный режим viewport
- `window/stretch/aspect="expand"` - расширение для заполнения экрана

## Адаптивность

### Динамическое позиционирование
- **Персонаж**: позиционируется на 20% от левого края и 70% от верха экрана
- **Земля**: находится на 70% от высоты экрана (адаптируется автоматически)
- **Препятствия и предметы**: спавнятся справа за экраном и движутся влево

### UI элементы
- Все UI элементы используют anchors для адаптивного позиционирования
- Кнопка JUMP центрирована внизу экрана
- TopBar (кнопки управления и счет) привязан к верхнему правому углу
- Элементы автоматически адаптируются при изменении ориентации

## Как это работает

1. При изменении размера viewport вызывается `_on_viewport_size_changed()`
2. Обновляются размеры экрана (`screen_width`, `screen_height`)
3. Пересчитывается позиция земли для всех объектов
4. Персонаж автоматически перемещается на новую позицию

## Тестирование

Для тестирования разных ориентаций:
1. В редакторе Godot: Project → Project Settings → Display → Window
2. Измените размер окна или ориентацию
3. Игра автоматически адаптируется

## Мобильные устройства

При экспорте на мобильные устройства:
- Игра будет работать в любой ориентации
- UI элементы автоматически адаптируются
- Позиционирование объектов корректно в обеих ориентациях
